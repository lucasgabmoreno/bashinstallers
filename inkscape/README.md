# Inkscape 1.1.1
![Portada](preview.png?raw=true)

## Install Inkscape:
```
git clone https://github.com/lucasgabmoreno/linuxmint-installers.git
cd linuxmint-installers/inkscape
bash install.sh
```

## Uninstall Inkscape:
```
git clone https://github.com/lucasgabmoreno/linuxmint-installers.git
cd linuxmint-installers/inkscape
bash uninstall.sh
```

## What this will install:
* inkscape
